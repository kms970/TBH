# 태스크바 히어로 자동 합성

Task Bar Hero 화면에서 버튼 이미지를 찾아 실제 마우스 클릭으로 상자 열기, 장비/재료 합성, 창고 이동을 진행하는 윈도우용 도우미입니다.

## 다운로드

최신 공유용 ZIP은 [GitHub Releases](https://github.com/kms970/TBH/releases/latest)에서 받으면 됩니다.

## 바로 사용하기

공유받는 사람에게는 [최신 릴리즈](https://github.com/kms970/TBH/releases/latest) 링크를 전달하세요.

1. ZIP 압축을 풉니다.
2. `TaskBarHeroAutoSynth.exe`를 실행합니다.
3. Task Bar Hero 게임 화면을 켭니다.
4. `인식 검사`로 주요 버튼이 잡히는지 확인합니다.
5. 반복 간격, 합성 레벨, 장비/재료 옵션을 고른 뒤 `자동 반복 시작` 또는 `한 번 실행`을 누릅니다.
6. 필요하면 `영역 다시 잡기`로 큐브/합성창, 창고/가방, 전투 목록/보상 상자가 모두 들어가게 드래그합니다. 잡은 영역은 창 기준으로 저장되어 창을 옮겨도 따라갑니다.

게임 설정의 `창 크기`가 `1x`, `1.25x`, `1.5x` 중 무엇인지 확인하고 프로그램의 `게임 창 크기` 드롭다운도 같은 값으로 맞추세요. 인식은 세 배율 템플릿을 함께 검사하므로 일부 PC에서 창 크기 설정이 달라도 버튼을 더 잘 찾습니다.

초록색 `ON` 토글은 켜진 상태이고, 회색 `OFF` 토글은 꺼진 상태입니다.

## 주요 동작

- 합성 UI가 안 보이면 메뉴 버튼과 큐브 버튼을 찾아 엽니다.
- 합성 전에 보상 상자 2종을 먼저 엽니다.
- 합성 레벨은 기본적으로 건드리지 않습니다. 게임의 자동 채우기가 아이템 레벨에 맞춰 합성 레벨을 조정합니다.
- `레벨 고정`을 켠 경우에만 선택한 합성 레벨을 맞춥니다.
- 장비와 재료 자동 채우기를 각각 더 이상 합성할 수 없을 때까지 반복합니다.
- 자동 채우기 후 합성이 되지 않으면 큐브에 들어간 아이템을 `돌아가기`로 빼고 다음 시도를 진행합니다.
- 합성 후 창고를 열고 `창고 < 가방`을 누릅니다.
- 정렬 후 인벤토리 첫 칸에 아이템이 남아 있으면 창고 페이지를 바꿔 다시 이동을 시도합니다.

## 화면 영역 기준

기본값은 `창 위치 추적 ON`입니다. 프로그램이 `TaskBarHero` 또는 `Task Bar Hero` 창을 찾으면 게임 창 전체를 검색 영역으로 사용하므로 처음 공유받은 PC에서도 별도 영역 설정 없이 시작할 수 있습니다.

창을 못 찾거나 특정 영역만 쓰고 싶을 때는 `영역 다시 잡기`를 누르세요. 창 위치 추적이 켜져 있으면 수동 영역도 게임 창 기준 상대좌표로 저장되어, 창을 옮겨도 같은 크기 영역을 따라갑니다.

전투 목록만 선택하면 큐브 버튼, 합성창, 창고/가방 버튼을 찾지 못할 수 있습니다. 수동 설정을 할 때는 큐브/합성창, 창고/가방, 전투 목록/보상 상자가 모두 포함되도록 넉넉히 잡으세요.

실행 중에는 실제 마우스 포인터가 클릭 위치로 이동합니다.

`창 위치 자동 추적`을 켜면 매 실행마다 `TaskBarHero` 창 위치를 다시 찾아서 창을 옮겨도 따라갑니다.

`창 앞으로 가져오기`를 켜면 실행 전에 게임 창을 앞으로 올립니다. 다른 창이 게임을 덮고 있으면 이미지 인식이 실패할 수 있기 때문에 기본값은 켜짐입니다.

Task Bar Hero에서는 포인터 이동 없는 Windows 메시지 클릭이 안정적으로 동작하지 않아 지원하지 않습니다. 실행 중에는 `일반 클릭` 방식으로 마우스 포인터가 자동 클릭 위치로 이동합니다.

게임의 `UI 레이아웃` 설정에 따라 영웅/큐브/창고 위치가 달라질 수 있습니다. 기본값인 `UI 위치 자동 탐색`을 켜두면 고정 좌표보다 버튼 이미지를 우선 찾아 클릭합니다.

프로그램은 게임의 `UI 레이아웃` 설정값을 강제로 바꾸지 않습니다. 대신 `상자 전 큐브 닫기` 옵션으로 큐브 창이 보상 상자를 가리는 상황을 줄입니다.

Windows가 디스플레이를 끄거나 잠금 화면으로 들어가면 화면 캡처와 마우스 입력이 실패할 수 있습니다. 장시간 돌릴 때는 전원 설정에서 화면 끄기/절전 시간을 충분히 길게 잡는 것을 권장합니다.

## 개발자용

소스 상태에서 실행하려면 Python 3.11 이상을 설치한 뒤:

```bat
install_deps.bat
pythonw TaskBarHeroAutoSynth.pyw
```

실행 파일을 다시 만들려면:

```bat
build_exe.bat
```

빌드 결과는 `dist\TaskBarHeroAutoSynth.exe`와 `release\TaskBarHeroAutoSynth_공유용.zip`에 생성됩니다.
