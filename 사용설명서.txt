태스크바 히어로 자동 합성 사용설명서

1. TaskBarHeroAutoSynth.exe를 실행합니다.
2. Task Bar Hero 게임 화면을 켭니다.
3. 인식 검사를 눌러 주요 버튼이 잡히는지 확인합니다.
4. 창을 자동으로 못 찾거나 버튼이 거의 잡히지 않을 때만 영역 다시 잡기를 누릅니다.
5. 화면이 반투명하게 덮이면 큐브/합성창, 창고/가방, 전투 목록/보상 상자가 모두 들어가게 드래그합니다.
6. 드래그를 놓으면 선택 영역이 게임 창 기준으로 저장됩니다. 창 위치 자동 추적은 계속 켜져 있어 창을 옮겨도 따라갑니다.
7. 반복 간격, 장비 합성, 재료 합성, 상자 먼저 열기 옵션을 고릅니다. 합성 레벨은 보통 건드리지 않아도 됩니다.
8. 자동 반복 시작 또는 한 번 실행을 누릅니다.

화면/레이아웃 설정:
- 게임 설정의 창 크기가 1x, 1.25x, 1.5x 중 무엇인지 확인하고 프로그램의 게임 창 크기도 같은 값으로 맞추세요.
- 프로그램은 1x, 1.25x, 1.5x 템플릿을 함께 검사해서 PC별 배율 차이를 보정합니다.
- 초록색 ON 토글은 켜진 상태이고, 회색 OFF 토글은 꺼진 상태입니다.
- 창 위치 자동 추적을 켜면 창을 옮겨도 매 실행마다 TaskBarHero 또는 Task Bar Hero 창 위치를 다시 찾습니다.
- 창 앞으로 가져오기를 켜면 실행 전에 게임 창을 앞으로 올립니다. 다른 창이 게임을 덮으면 이미지 인식이 실패할 수 있어 기본값은 켜짐입니다.
- UI 위치 자동 탐색을 켜면 게임의 UI 레이아웃 설정이 달라도 버튼 이미지를 우선 찾아 클릭합니다.
- 상자 전 큐브 닫기를 켜면 큐브 창이 보상 상자를 가리는 경우를 줄이기 위해 상자 확인 전에 큐브 창을 잠깐 닫습니다.
- Task Bar Hero에서는 포인터 이동 없는 클릭이 안정적으로 동작하지 않아 일반 클릭만 지원합니다.
- 프로그램은 사용자의 게임 UI 레이아웃 설정값을 강제로 변경하지 않습니다.

합성이 안 되는 경우:
- 자동 채우기 후 합성 버튼이 나오지 않으면 큐브에 들어간 아이템을 돌아가기 버튼으로 뺍니다.
- 돌아가기 버튼 인식이 실패하면 자동 채우기 버튼 기준의 예비 위치를 클릭해 한 번 더 비우기를 시도합니다.

화면 영역 기준:
- 화면 영역은 자동으로 확장되지 않습니다.
- 프로그램은 사용자가 지정한 사각형 안에서만 버튼과 상자를 찾습니다.
- 기본값은 창 위치 자동 추적 ON이므로 처음에는 수동 영역 없이 인식 검사부터 해보세요.
- 전투 목록만 잡으면 큐브 버튼, 합성창, 창고/가방 버튼을 찾지 못할 수 있습니다.
- 수동 영역을 잡을 때는 큐브/합성창, 창고/가방, 전투 목록/보상 상자가 모두 포함되게 넉넉히 잡으세요.

합성 레벨:
- 기본값은 레벨 고정 OFF입니다.
- 게임의 자동 채우기가 아이템 레벨에 맞춰 합성 레벨을 조정하므로 보통은 그대로 두세요.
- 레벨 고정을 켠 경우에만 GUI에서 선택한 합성 레벨을 맞춥니다.
- 선택한 레벨이 잠겨 있으면 가장 가까운 하위 열린 레벨로 자동 변경합니다.

인식이 안 될 때:
- 게임 창이 다른 창에 가려져 있지 않은지 확인하세요.
- 인식 검사 후 logs 폴더의 last_scan.png를 확인하세요.
- 빨간 박스가 거의 없으면 게임 창 크기 값을 바꿔 다시 검사하세요.
- 창 자동 추적이 실패하면 영역 다시 잡기를 사용하세요.
- Windows 화면 배율이 너무 다르면 인식률이 떨어질 수 있습니다.

주의:
- 실행 중에는 마우스 포인터가 자동 클릭 위치로 이동합니다.
- 마우스를 쓰는 다른 작업과 동시에 실행하지 않는 것이 좋습니다.
- Windows가 디스플레이를 끄거나 잠금 화면으로 들어가면 캡처/클릭이 실패할 수 있습니다.
- 게임 화면 배치가 바뀌면 인식 검사 후 필요할 때 영역 다시 잡기를 다시 해야 합니다.
