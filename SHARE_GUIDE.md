# 공유용 사용 설명서

이 프로그램은 Task Bar Hero 화면을 이미지로 인식해서 실제 마우스 클릭으로 자동 합성을 진행합니다. 기본값은 게임 창 자동 추적이라, 공유받은 PC에서도 Task Bar Hero 창을 켠 뒤 바로 인식 검사부터 해볼 수 있습니다.

## 1. 공유할 파일

공유할 때는 `release\TaskBarHeroAutoSynth_공유용.zip`만 보내면 됩니다.

ZIP 안에는 다음 파일이 들어갑니다.

- `TaskBarHeroAutoSynth.exe`
- `사용설명서.txt`

소스 코드, Python 파일, 템플릿 폴더를 따로 보낼 필요가 없습니다. 버튼 이미지 템플릿은 실행 파일 안에 포함됩니다.

## 2. 처음 실행

1. ZIP 압축을 풉니다.
2. `TaskBarHeroAutoSynth.exe`를 실행합니다.
3. Task Bar Hero 게임 화면을 엽니다.
4. `인식 검사`를 눌러 버튼 인식 상태를 확인합니다.
5. 창을 찾지 못하거나 버튼이 거의 잡히지 않을 때만 `영역 다시 잡기`를 누릅니다.
6. 화면이 반투명하게 덮이면 큐브/합성창, 창고/가방, 전투 목록/보상 상자가 모두 들어가게 드래그합니다.
7. 드래그를 놓으면 선택 영역이 게임 창 기준으로 저장됩니다. 창 위치 추적은 계속 켜져 있어 창을 옮겨도 따라갑니다.

수동 화면 영역의 크기는 자동으로 확장되지 않습니다. 전투 목록만 선택하면 큐브 버튼, 합성창, 창고/가방 버튼을 찾지 못할 수 있습니다. 게임 UI, 큐브/합성창, 창고/가방, 전투 목록/보상 상자가 모두 들어가게 넉넉히 잡으세요.

## 3. GUI 설정

- `반복 간격`: 자동 반복 실행 간격입니다. 단위는 분입니다.
- `합성 레벨`: `레벨 고정`을 켰을 때만 사용할 합성 레벨 범위입니다.
- `레벨 고정`: 보통은 꺼두세요. 게임의 자동 채우기가 아이템 레벨에 맞춰 합성 레벨을 조정합니다.
- `상자 먼저 열기`: 합성 전에 보상 상자 2종을 먼저 엽니다.
- `장비 합성`: 장비 자동 채우기 합성을 진행합니다.
- `재료 합성`: 재료 자동 채우기 합성을 진행합니다.
- 초록색 `ON`으로 채워진 토글은 켜진 상태이고, 회색 `OFF`는 꺼진 상태입니다.
- `게임 창 크기`: 게임 설정의 `창 크기` 값입니다. 게임이 `1.25x`면 프로그램도 `1.25x`로 맞추세요. 프로그램은 1x, 1.25x, 1.5x 템플릿을 함께 검사해서 배율 차이를 보정합니다.
- `창 위치 자동 추적`: 매 실행마다 `TaskBarHero` 창 위치를 다시 찾아 창을 옮긴 사용자도 대응합니다.
- `창 앞으로 가져오기`: 실행 전에 게임 창을 앞으로 올립니다. 다른 창이 게임을 덮으면 이미지 인식이 실패할 수 있어 기본값은 켜짐입니다.
- `UI 위치 자동 탐색`: 게임의 `UI 레이아웃` 설정이 달라도 버튼 이미지를 우선 찾아 클릭합니다.
- `상자 전 큐브 닫기`: 큐브 창이 보상 상자를 가릴 수 있어, 상자를 찾기 전에 큐브 창을 잠깐 닫습니다.
- `클릭 방식`: Task Bar Hero에서는 `일반 클릭`만 지원합니다. 실행 중에는 마우스 포인터가 자동 클릭 위치로 이동합니다.

프로그램은 사용자의 게임 `UI 레이아웃` 설정값을 강제로 변경하지 않습니다.
- `자동 반복 시작`: 설정한 간격마다 계속 실행합니다.
- `이번만 실행`: 지금 한 번만 실행합니다.
- `중지`: 다음 반복 실행을 멈춥니다.

## 4. 레벨 고정과 잠긴 레벨 처리

기본값은 `레벨 고정 OFF`입니다. 이 상태에서는 프로그램이 합성 레벨 드롭다운을 건드리지 않고, 게임의 자동 채우기가 아이템 레벨에 맞춰 처리합니다.

`레벨 고정`을 켠 경우에만 GUI의 합성 레벨 선택값을 사용합니다. 선택한 레벨에 자물쇠가 보이면, 프로그램은 그 PC에서 아직 잠긴 레벨로 판단하고 가장 가까운 하위 열린 레벨로 자동 변경합니다.

예를 들어 `Lv.40~65`를 선택했는데 해당 PC에서 잠겨 있으면 `Lv.30~50`, 그것도 잠겨 있으면 `Lv.20~40` 순서로 내려갑니다. 다른 PC에서 `Lv.40~65`가 열려 있으면 그대로 선택합니다.

## 5. 자동 진행 순서

1. 합성 UI가 이미 보이는지 확인합니다.
2. 안 보이면 메뉴 버튼과 큐브 버튼을 눌러 합성 UI를 엽니다.
3. 보상 상자 2종을 모두 엽니다.
4. 선택한 합성 레벨을 맞춥니다.
5. 장비 자동 채우기 후 합성을 반복합니다.
6. 재료 자동 채우기 후 합성을 반복합니다.
7. 자동 채우기 후 합성이 되지 않으면 큐브에 들어간 아이템을 `돌아가기`로 빼고 다음 시도를 진행합니다.
8. 창고를 열고 `창고 < 가방`을 누릅니다.
9. 가방을 정렬한 뒤 첫 칸에 아이템이 남아 있으면 창고 페이지를 바꿔 다시 `창고 < 가방`을 누릅니다.

## 6. 인식이 안 될 때

1. 게임 창이 다른 창에 가려져 있지 않은지 확인합니다.
2. `인식 검사`를 누르고 `logs\last_scan.png`를 확인합니다.
3. 빨간 박스가 거의 없으면 `게임 창 크기` 값을 바꿔 다시 검사합니다.
4. 창을 자동으로 못 찾으면 `영역 다시 잡기`를 사용합니다.
5. Windows 화면 배율, 게임 UI 스킨, 언어가 다르면 인식률이 떨어질 수 있습니다.

## 7. 실행 중 주의

- 실행 중에는 마우스 포인터가 클릭 위치로 이동합니다.
- 마우스를 쓰는 다른 작업과 동시에 실행하지 않는 것이 좋습니다.
- 게임 화면 위에 다른 창을 올리면 클릭이 실패할 수 있습니다.
- Windows가 디스플레이를 끄거나 잠금 화면으로 들어가면 캡처/클릭이 실패할 수 있습니다.
- 프로그램은 이미지 인식으로만 동작하므로, 화면 배치가 바뀌면 `인식 검사` 후 필요할 때 `영역 다시 잡기`를 다시 해야 합니다.
